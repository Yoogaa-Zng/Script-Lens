# 劇本結構分析工作台

McKee 事件戲／狀態戲分類的互動工作台。純前端單一 HTML 檔案，不需安裝、不需伺服器，開啟 `index.html` 即可使用；部署到 GitHub Pages 後即為雲端網址，手機也能開。

## 使用方式
- **工作台**分頁：編輯場組、角色弧線、角色分析、台詞比重、主題，圖表即時更新。
- **報告預覽**分頁：六節報告的完整呈現。
- **轉出 PDF**：切到報告預覽並開啟瀏覽器列印對話框，選「另存為 PDF」。
- **匯出 Python 資料結構**：把目前資料轉成可直接貼進《劇本分析報告生成器_v3.4.py》的字面量，用於產出正式高品質 PDF（含完整繪圖細節、STSong-Light 中文字體）。

## 定位
這個網頁工具負責「整理資料、即時檢查、快速預覽」。正式交付品質的 PDF 仍建議用 Python 生成器產出，兩者資料結構完全對應，可用「匯出」功能無縫銜接。
